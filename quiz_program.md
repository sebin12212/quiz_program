# Quiz Program

## 실행 방법

1. 저장소를 다운로드합니다.
2. 터미널에서 해당 폴더로 이동합니다.
3. 아래 명령어로 컴파일 및 실행합니다.

```bash
javac Main.java
java Main
```

---

## 프로젝트 설명

Java를 활용하여 **콘솔 기반 퀴즈 프로그램**을 구현한 프로젝트입니다.

이산수학 및 Linux 관련 문제를 출제하고,  
사용자가 정답을 입력하면 즉시 피드백과 함께 최종 점수를 출력합니다.

---

## 주요 기능

- 퀴즈 문제 랜덤 출제
- 사용자 정답 입력 및 즉시 피드백
- 최종 점수 계산 및 출력

| 기능 | 설명 |
|------|------|
| 문제 출제 | 이산수학 / Linux 관련 퀴즈 |
| 정답 확인 | 정답 / 오답 즉시 출력 |
| 점수 계산 | 전체 문제 중 맞춘 개수 출력 |

---
